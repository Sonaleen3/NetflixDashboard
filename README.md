# NetflixDashboard

TITLE : Netflix Dashboard


DESCRIPTION :
The Netflix Dashboard is an interactive data visualization project designed to analyze and explore Netflix’s content library using multiple visual elements. The dashboard provides insights into content distribution, genres, release trends, and type of content available on the platform.

Slicers are used to enable dynamic filtering of the dataset, allowing users to easily view specific information based on categories such as year, genre, or content type. A line chart is used to display trends in Netflix content releases over time, helping identify growth patterns and changes in production across different years.

A donut chart visualizes the proportion of Movies versus TV Shows available on the platform, providing a quick understanding of the content type distribution. The treemap highlights the most common genres and their relative share in the Netflix catalog, making it easier to compare category sizes visually.

Additionally, a stacked bar chart is used to analyze content distribution across multiple categories, such as genre by type or content by country, allowing viewers to see both total values and their composition in a single chart.

Overall, this dashboard transforms raw Netflix data into meaningful insights through interactive visuals, enabling users to quickly understand content trends, distribution, and patterns within the platform’s catalog.
